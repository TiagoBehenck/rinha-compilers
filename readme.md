# Rinha de compiladores 

Criei um interpretador em Javascript, apenas entende o comando Print, para exercitar os conceitos que sรณ vi na faculdade. =) 

## Como fazer funcionar 

Primeiramente clonar o projeto 

```git clone https://github.com/TiagoBehenck/rinha-compilers.git```

### No terminal

Rodar o terminal (precisa do Node.js instalado)

```node execute.js```

### Com Docker

```docker build -t rinha .```
```docker run rinha```
